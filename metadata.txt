    configurations (perspectives) in QGIS.
    Switch between workspaces in one click.
about=The QWorkspaceSwitcher plugin allows QGIS
    users to create, save and apply named interface
    configurations called perspectives. Each
    perspective defines which panels, toolbars and
    menus are visible and how they are organized.
    Third-party plugins can declare their own
    perspectives via a .psp.json file placed at
    the root of their plugin folder.
